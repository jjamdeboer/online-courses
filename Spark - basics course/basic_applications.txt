SPARKCONTEXT (CREATED OBJECT WITHIN SPARK APPLICATION) MAIN ENTRY POINT IN SPARK: OBJECT THAT REPRESENTS THE CONNECTION TO A SPARK CLUSTER
CAN BE USED FOR RDD'S AND SHARED VARIABLES (sc IN THE SPARK SHELL)

SPARKCONF-FILE CONTAINS THE MAIN INFORMATION ABOUT THE APPLICATION AND PASS THAT TO THE SPARKCONTEXT

- ANONYMOUS FUNCTIONS/LAMBDA EXPRESSIONS
- STATIC METHODS IN A GLOBAL SINGLETON OBJECT
- PASS REFERENCE TO A METHOD IN A CLASS INSTANCE
