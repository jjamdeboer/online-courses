TO SEE CHANGES BETWEEN TWO COMMITS, USE: git diff [COMMIT ID 1] [COMMIT ID 2]
TO SEE CHANGES BETWEEN TWO STATES, SUCH AS THE STAGING AREA AND THE HEAD COMMIT OR THE WORKING DIRECTORY AND THE HEAD COMMIT WHEN NOT STAGED YET (DEFAULT): git diff --staged/git diff
TO SEE CHANGES BETWEEN A FILE BETWEEN TWO COMMITS: git diff [COMMIT ID 1] [COMMIT ID 2] [NAME FILE]
