# THIS DOCUMENT CONTAINS CRITIQUE ON SQL/POSTGRES-SYNTAX
- ALIASES OFTEN NOT REUSABLE ACROSS WHOLE QUERY
- OFTEN NO LEADING COMMA'S ALLOWED
- OFTEN NO QUALIFY-STATEMENT
- NOT ALL CONDITIONS CAN BE PUT IN WHERE-CLAUSE, WHICH SHOULD BE STANDARD
- NO STANDARD WITH REGARDS TO INDENTATION AND STYLE
- GROUPING BY SHOULD BE ON THE NON-AGGREGATED COLUMNS AS A STANDARD (SHOULDN'T BE NEEDED TO MAKE EXPLICIT EVERY TIME)
- SELECT * EXCEPT... SHOULD BE EXCEPTED SYNTAX
